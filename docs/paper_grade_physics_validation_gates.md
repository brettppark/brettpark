# Paper-Grade Physics Validation Gates

<span style="color:red"><b>Current status:</b> this file is a gate list, not a
claim that the present AREPO branch already produces publishable SN Ia
explosions. A module moves from development to paper-use only after the matching
validation artifacts below exist in `production_runs/`, `results/figures/`, and
the run log.</span>

## Gate Logic

Each stage is tracked separately:

| Stage | Meaning | Paper status |
|---|---|---|
| Implemented | Code path compiles and runs on a controlled case. | Engineering only. |
| Verified | Code path passes a targeted benchmark or conservation test. | Methods appendix candidate. |
| Science-ready | Code path passes resolution, physical benchmark, and provenance checks. | Can be used for paper figures/tables. |

## AREPO Stellar/SN Ia Physics Gates

| Physics item | Current branch status | Required validation artifact before paper use |
|---|---|---|
| Self-gravity hydro | AREPO tree gravity plus finite-volume gas hydro are active in isolated-WD and binary runs. | Isolated WD mass, energy, angular momentum, center-of-mass, and radial profile conservation over multiple dynamical times. |
| WD equation of state | `SNIA_HELMHOLTZ_EOS` is a Helmholtz-like degeneracy/thermal/radiation proxy. | Replace or calibrate against a full Timmes/Swesty Helmholtz table; produce one-zone comparisons versus MESA profile thermodynamics across WD density/temperature/composition. |
| MESA profile bridge | Isolated-WD IC creation can map a `profile*.data` density, temperature, and composition profile. | Plot input MESA profile against IC radial bins at `t=0` and after relaxation; quantify mass and composition errors. |
| Numerical atmosphere | Passive-scalar atmosphere clamp prevents low-density background cells from being treated as stellar gas. | Atmosphere-density resolution study showing WD surface mass loss and scalar contamination converge. Stellar radiation feedback is not the correct control for this numerical atmosphere. |
| Isolated WD relaxation | Analytic Lane-Emden and MESA-profile isolated-WD paths exist. | Resolution ladder and time ladder: at least three cell counts and several dynamical times, with tolerances stated in the paper. |
| Roche/contact diagnostics | Binary module writes separation, COM, Roche-lobe estimates, and component/fuel diagnostics. | Validate Roche-lobe filling against the source binary state; show mass flux through L1 and angular momentum conservation after the artificial driver is disabled. |
| Inspiral driver | Artificial inspiral hooks exist for development, off in isolated-WD runs. | Calibrate against a documented angular-momentum-loss prescription or start directly at contact; prove the driver is off before physical mass transfer/explosion claims. |
| Nuclear burning | Reduced-alpha source-term hook exists but is disabled in relaxation runs. | One-zone burn tests, energy conservation, timestep-limiter tests, and a resolved planar detonation benchmark. |
| Detonation criterion | Hotspot/criterion diagnostics exist; no validated front propagation. | Cited critical hotspot/edge-lit criterion, resolution requirement, event metadata, and a controlled detonation benchmark before WD-scale detonation. |
| SD deflagration/DDT | SD branch writes flame/turbulence/DDT diagnostics only. | ADR or level-set flame, turbulent flame speed model, and calibrated DDT criterion. This is later than DD/D6 in the implementation order. |
| Tracer/isotope export | Passive scalar channels and diagnostic export exist. | Lagrangian tracer histories or validated isotope-binned ejecta export suitable for ARTIS/TARDIS/SEDONA. |
| Observables | Not generated by AREPO. | Homologous ejecta -> RT light curves/spectra -> sncosmo/SALT3 fit yielding `M_B`, `x1`, `c`, with fitter metadata. |

## Present Paper-Use Boundary

The current runs can support a methods-development statement:

```text
We built a source-traced MESA-to-AREPO bridge and validated that the workflow
can create and evolve isolated WD initial conditions with self-gravity,
degenerate-EOS hooks, and numerical-atmosphere control.
```

The current runs cannot yet support:

```text
We simulated a physical CO+CO merger, D6 explosion, Chandrasekhar DDT
explosion, Ni56 yield, or SALT3 observable.
```
